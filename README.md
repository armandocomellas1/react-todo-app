- Code editor
- Knowledge in HTML

- Clone the repository

- Install dependencies
- Open project with text editor to view changes
- Run the code using [live server](https://www.google.com/search?client=safari&rls=en&q=live+server&ie=UTF-8&oe=UTF-8)
